Glenn's Game of Life

Keys:
s: Start/Stop Simulation
d: Delete mode on/off (for mouse dragging)
c: Clears all the cells
r: Creates a random grid

Pressing 1 through 9 also loads some predefined layouts
1: Slider
2: Ten in a Row
3: Exploder
4: Small Exploder
5: Fish
6: Fountain
7: Just a line, but creates a decent effect when run
8: A Greeting
9: Java
